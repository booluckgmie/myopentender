The following Malaysian tender portals are tracked. Connect a scraper backend (Python requests + BeautifulSoup / Playwright) to auto-populate the SQLite database.

Scraper Notes
  • Most portals require Playwright / Selenium for JS-rendered pages (ePerolehan, SPSB, Prasarana)
  • Sarawak & Perak S3PK expose plain HTML tables — use requests + BeautifulSoup
  • Bursa: scrape company_announcement page, filter announcements containing "tender"
  • Store normalized rows in SQLite: tenders table (schema matches this app)
  • Export as tenders.sqlite and load via sql.js FileReader API
  • Run scraper on cron (daily at 08:00) and push updated DB to the same GitHub Pages deployment
